# PDF RAG Chatbot

A Retrieval-Augmented Generation (RAG) chatbot that reads PDF documents and answers questions about them using LangChain, ChromaDB, and Gradio.

## Features
- PDF uploading
- Text splitting and embedding
- Vector database (Chroma)
- Retriever-based context search
- LLM-powered question answering
- Gradio web interface

## How it works
1. Load a PDF
2. Split text into chunks
3. Embed chunks into a vector database
4. Retrieve relevant context
5. Generate an answer using an LLM
